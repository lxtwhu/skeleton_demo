# 项目简介
## 项目结构

```
.
├── build/ (聚合子项目，打包需要)
├── build.json (项目室容器化编译配置)
├── demo-java-ac/ (一个使用AC容器的java子项目)
├── demo-java-static/ (一个静态java子项目)
├── demo-java-upjas/ (一个使用UPJAS容器的java子项目)
├── demo-local-component/ (一个作为依赖的本地组件子项目)
├── demo-resource-no-setup/ (一个无需安装的资源子项目)
├── demo-resource-static/ (一个静态资源子项目)
├── .git/ (git本地仓库)
├── .gitignore (git忽略文件配置)
├── pom.xml (父项目的pom文件)
└── README.md (项目介绍文档)
```
